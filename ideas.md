# 5가지 사랑의 언어 진단 - 디자인 브레인스토밍

## 세 가지 접근 방식

### 1. "Warm Embrace" - 따뜻한 포옹
- **소개**: 부드러운 곡선과 따뜻한 파스텔 톤을 활용한 감성적 디자인. 사랑과 관계라는 주제에 걸맞은 포근하고 친밀한 느낌을 전달한다.
- **확률**: 0.04

### 2. "Constellation" - 별자리
- **소개**: 어두운 배경 위에 빛나는 별자리처럼 5가지 사랑의 언어를 시각화. 각 언어가 하나의 별처럼 빛나며, 사용자의 고유한 사랑의 패턴을 우주적 스케일로 표현한다.
- **확률**: 0.03

### 3. "Paper & Ink" - 종이와 잉크
- **소개**: 수채화 텍스처와 손글씨 느낌의 타이포그래피를 결합한 아날로그 감성 디자인. 편지를 쓰는 듯한 따뜻하고 개인적인 경험을 제공한다.
- **확률**: 0.07

## 선택된 접근 방식: "Warm Embrace" - 따뜻한 포옹

### Design Movement
Organic Modernism + Emotional Design — 유기적 곡선과 현대적 미니멀리즘의 결합. 감정을 시각적으로 전달하는 데 초점을 맞춘 따뜻한 모더니즘.

### Core Principles
1. **유기적 곡선**: 모든 요소에 부드러운 곡선과 자연스러운 형태를 사용하여 딱딱함을 배제
2. **감정적 색채**: 각 사랑의 언어에 고유한 색상을 부여하여 직관적 인식 유도
3. **점진적 공개**: 설문 진행 과정을 단계적으로 보여주며 몰입감 있는 여정 제공
4. **개인화된 결과**: 결과 페이지가 사용자의 고유한 프로필처럼 느껴지도록 설계

### Color Philosophy
사랑의 5가지 언어 각각에 감정적 의미를 담은 색상 체계:
- **인정하는 말 (A)**: Coral Rose `#E8736F` — 따뜻한 말의 온기
- **함께하는 시간 (B)**: Amber Gold `#F5A623` — 함께하는 시간의 소중함
- **선물 (C)**: Sage Green `#7FB069` — 선물에 담긴 정성
- **봉사 (D)**: Soft Lavender `#9B8EC4` — 헌신의 고귀함
- **스킨십 (E)**: Warm Peach `#F4845F` — 스킨십의 따뜻함
- **배경**: Cream White `#FDF8F4` — 포근한 크림색 배경
- **텍스트**: Warm Charcoal `#3D3535` — 부드러운 다크 톤

### Layout Paradigm
**Flowing Card Journey** — 전통적 그리드를 벗어나 흐르는 듯한 카드 기반 레이아웃. 설문은 한 문항씩 풀스크린으로 표시되며, 진행 상황은 유기적 곡선 프로그레스 바로 표현. 결과 페이지는 비대칭 레이아웃으로 시각적 흥미를 유발.

### Signature Elements
1. **물결 프로그레스 바**: 직선이 아닌 부드러운 물결 형태의 진행 표시
2. **하트 레이더 차트**: 결과를 5각형 레이더 차트로 시각화하되, 꼭짓점을 하트 모양으로 장식
3. **그라데이션 카드**: 선택지 카드에 호버 시 해당 언어의 색상으로 부드럽게 전환되는 그라데이션

### Interaction Philosophy
모든 인터랙션은 "포옹"처럼 부드럽고 따뜻해야 한다. 클릭 시 살짝 눌리는 듯한 스케일 효과, 전환 시 페이드와 슬라이드의 조합, 결과 공개 시 점진적 애니메이션으로 기대감 조성.

### Animation
- 페이지 전환: 300ms ease-out slide + fade
- 카드 호버: 200ms scale(1.02) + shadow 증가
- 카드 선택: 160ms scale(0.97) → scale(1.0) + 색상 전환
- 결과 차트: 800ms staggered reveal (각 축 100ms 간격)
- 프로그레스: 500ms cubic-bezier(0.23, 1, 0.32, 1) 물결 애니메이션

### Typography System
- **Display/Heading**: "Noto Serif KR" — 우아하고 따뜻한 세리프체
- **Body**: "Pretendard" — 가독성 높은 현대적 산세리프
- **Accent**: 숫자와 강조에 "Noto Serif KR" italic 활용
- 계층: Display 2.5rem / H1 2rem / H2 1.5rem / Body 1rem / Caption 0.875rem

### Brand Essence
"당신의 사랑을 이해하는 첫 걸음" — 커플, 가족, 연인을 위한 관계 진단 도구. 따뜻하면서도 과학적인 접근으로 차별화.
성격 형용사: **따뜻한, 신뢰할 수 있는, 개인적인**

### Brand Voice
헤드라인과 CTA는 마치 친한 친구가 다정하게 말하는 듯한 톤. 전문적이되 딱딱하지 않고, 개인적이되 가벼지 않은 목소리.
- 예시 1: "당신은 어떤 사랑의 언어로 대화하고 있나요?"
- 예시 2: "30개의 질문으로 발견하는, 나만의 사랑 표현법"

### Wordmark & Logo
하트 형태를 추상화한 그래픽 심볼. 5개의 부드러운 곡선이 하나의 하트를 이루는 형태로, 5가지 언어의 조화를 상징. 각 곡선은 고유 색상을 가짐.

### Signature Brand Color
**Coral Rose `#E8736F`** — 사랑의 따뜻함과 생동감을 동시에 전달하는 코랄 로즈. 너무 강렬하지 않으면서도 눈에 띄는 브랜드 컬러.
